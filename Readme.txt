Projeto Jogo das Capitais
## 📌 Sobre
Jogo das Capitais é um jogo educacional interativo desenvolvido com tecnologias web, que desafia o jogador a identificar as capitais dos estados brasileiros. Ao iniciar o jogo, o jogador insere seu nome e responde a uma sequência de perguntas com múltiplas escolhas. O sistema exibe o número de acertos e erros e permite reiniciar após completar todas as perguntas.

- 🧠 Perguntas dinâmicas sobre as capitais do Brasil.
- 🟢 Contador de acertos e erros atualizado em tempo real.
- 🔁 Opção para reiniciar o jogo após completar todas as rodadas.
- 🎨 Interface simples e responsiva.
- 👤 Identificação personalizada com nome do jogador.

##🛠 Tecnologias Utilizadas

Tecnologia
- 🎨 HTML -> Estrutura da interface
- 🎨 CSS -> Estilização visual
- 🟨 JavaScript -> Lógica do jogo e manipulação do DOM
